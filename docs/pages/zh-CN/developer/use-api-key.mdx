# 创建 API 密钥

您需要 API 密钥才能与 [Zeabur 的开放 API](./public-api) 互动。

## 创建 API 密钥

打开 [Zeabur Dashboard > 设置 > API 密钥](https://zeabur.com/account/api-keys)，即可看到当前生效的所有密钥。

![开发者面板](/developer/api-key/developer.en-US.png)

点击"生成新的 API 密钥"，就会以当前的时间生成一组新的 API 密钥。

![生成 API 密钥](/developer/api-key/generate-api-key.en-US.png)

点击"眼睛"按钮，即可查看 API 密钥。

![查看 API 密钥](/developer/api-key/view-api-key.en-US.png)

点击"复制"按钮，即可复制 API 密钥。

![复制 API 密钥](/developer/api-key/copy-api-key.en-US.png)

## 撤销 API 密钥

打开 [Zeabur Dashboard > 设置 > API 密钥](https://zeabur.com/account/api-keys)，即可看到当前生效的所有密钥。点击"删除"按钮，就会直接撤销这个 API 密钥。

## 使用 API 密钥

在您想调用 API 的 `Authorization` Header 中传入 `Bearer API_TOKEN`，示例如下：

```bash
curl --request POST \
  --url https://api.zeabur.com/graphql \
  --header 'Authorization: Bearer {YOUR_API_TOKEN}' \
  --header 'Content-Type: application/json' \
  --data '{"query":"query { me { username } }"}'
```
