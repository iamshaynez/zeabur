# Create API Keys

You need an API key to interact with [Zeabur's Public API](./public-api).

## Creating an API Key

Open [Zeabur Dashboard > Settings > API Keys](https://zeabur.com/account/api-keys) to see all currently active keys.

![Developer Panel](/developer/api-key/developer.en-US.png)

Click "Generate new API key" to create a new API key with the current timestamp.

![Generate API Key](/developer/api-key/generate-api-key.en-US.png)

Click the "eye" button to view the API key.

![View API Key](/developer/api-key/view-api-key.en-US.png)

Click the "copy" button to copy the API key.

![Copy API Key](/developer/api-key/copy-api-key.en-US.png)

## Revoking an API Key

Open [Zeabur Dashboard > Settings > API Keys](https://zeabur.com/account/api-keys) to see all currently active keys. Click the "delete" button to immediately revoke the API key.

## Using an API Key

Pass `Bearer API_TOKEN` in the `Authorization` header when calling the API, as shown in the example below:

```bash
curl --request POST \
  --url https://api.zeabur.com/graphql \
  --header 'Authorization: Bearer {YOUR_API_TOKEN}' \
  --header 'Content-Type: application/json' \
  --data '{"query":"query { me { username } }"}'
```
