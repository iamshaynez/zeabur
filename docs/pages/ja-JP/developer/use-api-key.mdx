# API キーの作成

[Zeabur のオープン API](./public-api) とやり取りするには、API キーが必要です。

## API キーの作成

[Zeabur Dashboard > 設定 > API キー](https://zeabur.com/account/api-keys)を開くと、現在有効なすべてのキーを確認できます。

![開発者パネル](/developer/api-key/developer.en-US.png)

「新しい API キーを生成」をクリックすると、現在の時刻に基づいて新しい API キーが生成されます。

![API キーの生成](/developer/api-key/generate-api-key.en-US.png)

「目」ボタンをクリックすると、API キーを確認できます。

![API キーの確認](/developer/api-key/view-api-key.en-US.png)

「コピー」ボタンをクリックすると、API キーをコピーできます。

![API キーのコピー](/developer/api-key/copy-api-key.en-US.png)

## API キーの取り消し

[Zeabur Dashboard > 設定 > API キー](https://zeabur.com/account/api-keys)を開くと、現在有効なすべてのキーを確認できます。「削除」ボタンをクリックすると、その API キーが直ちに取り消されます。

## API キーの使用

API を呼び出す際に、`Authorization` ヘッダーに `Bearer API_TOKEN` を含めます。以下は例です：

```bash
curl --request POST \
  --url https://api.zeabur.com/graphql \
  --header 'Authorization: Bearer {YOUR_API_TOKEN}' \
  --header 'Content-Type: application/json' \
  --data '{"query":"query { me { username } }"}'
```
