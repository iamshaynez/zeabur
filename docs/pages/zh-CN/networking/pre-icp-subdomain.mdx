# 预备案子域名

根据中国大陆法律法规，所有位于中国大陆境内的服务器部署的服务必须完成 ICP 备案才可以访问，这对于某些还处于开发阶段的项目来说可能会造成一些困扰。

在传统的做法中，如果你购买了中国大陆境内的服务器，在没有备案的情况下你可以使用 IP + 端口号的方式来访问服务，但这个做法导致你无法使用域名来访问服务，也没办法签发 SSL 证书。

对于这类情况，Zeabur 提供了预备案子域名的功能，让你可以在完成了实名认证的前提下，暂时使用 Zeabur 提供的已备案域名来访问你部署的服务，这对于开发阶段或是仅用于测试的个人临时服务来说非常方便，且完全合法合规。

## 使用条件

- 你的服务器必须是从 Zeabur 购买的。
- 你的服务器必须在中国大陆境内。
- 你的 Zeabur 账号必须已经完成了中国大陆身份实名认证。
- 你的服务器供应商必须已经在 Zeabur 支持预备案子域名:

| 云服务商 | 有 Zeabur 共享集群 | 可购买 Zeabur 托管服务器 | 有中国大陆区域 | 支持预备案子域名 |

| 阿里云 | ❌ | ✅ | ✅ | ✅ |
| 腾讯云 | ❌ | ✅ | ✅ | ✅ |

## 使用方法

1. 选择你部署在中国大陆境内的 Zeabur 托管服务器的项目，然后选择想要绑定预备案子域名的服务。
2. 选择「网络」标签页。
3. 点击「生成域名」，输入你想要的子域名。

![](https://cdn.zeabur.com/pre-icp-1.png)

4. 点击「创建域名」，如果你还没有完成实名认证，Zeabur 会提示你完成实名认证。
5. 在实名认证窗口输入你的姓名及身份证号，点击提交就会自动跳转到阿里云的实名认证服务。

![](https://cdn.zeabur.com/pre-icp-2.png)

6. 完成实名认证后，重新点击「生成域名」，就可以自动生成并绑定已备案的域名。

## 注意事项

- 使用预备案子域名即表示你已经同意对应云服务商的使用条款。
- Zeabur 严格禁止使用预备案子域名进行任何违反当地法律法规的行为，一旦发现 Zeabur 有权单方面终止服务并将实名信息移交用户所在国家/地区的执法机构。
- 不建议使用预备案子域名作为商用服务的正式域名，若测试/开发完成，建议进行正式的备案流程申请您自己的域名，Zeabur 免费提供备案申请的服务及支持。
